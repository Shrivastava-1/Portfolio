<h1>🌐 Personal Portfolio Website</h1>

<p>This is a <strong>responsive portfolio website</strong> showcasing my skills, experience, and projects. It provides insights into my background, expertise in <strong>Data Science, Python, SQL</strong>, and my latest works. The website also includes a <strong>contact form</strong> for easy communication.</p>

<h2>✨ Features</h2>
<ul>
  <li>✅ <strong>Homepage</strong> – Introduction with a professional greeting.</li>
  <li>✅ <strong>About Me</strong> – My skills, education, and work experience.</li>
  <li>✅ <strong>Services</strong> – Details of my expertise in <strong>Data Science, Python, and SQL</strong>.</li>
  <li>✅ <strong>Portfolio</strong> – Showcasing projects with descriptions and GitHub links.</li>
  <li>✅ <strong>Contact</strong> – Contact form for inquiries and social media links.</li>
  <li>✅ <strong>Responsive Design</strong> – Works on all screen sizes.</li>
</ul>

<h2>🛠️ Technologies Used</h2>
<ul>
  <li><strong>HTML, CSS</strong> – For website structure and styling.</li>
  <li><strong>JavaScript</strong> – For interactive elements like tabs and menu.</li>
  <li><strong>FontAwesome Icons</strong> – Used for icons throughout the website.</li>
  <li><strong>Google Fonts</strong> – Styling with the "Poppins" font.</li>
  <li><strong>Google Sheets API</strong> – Handles form submissions.</li>
</ul>

<h2>📂 Project Structure</h2>
<pre>
/portfolio-website  
│── index.html       # Main landing page  
│── style.css        # Styling file  
│── images/          # Folder containing images (profile, projects, etc.)  
│── scripts.js       # JavaScript for interactive features  
│── README.md        # Project documentation  
</pre>

<h2>🚀 How to Use</h2>

<h3>1️⃣ Clone the Repository</h3>
<pre>
git clone https://github.com/your-username/portfolio-website.git
cd portfolio-website
</pre>

<h3>2️⃣ Open <code>index.html</code> in a Browser</h3>
<h3>3️⃣ Customize the Content</h3>
<ul>
  <li>Replace text with your own details in <code>index.html</code>.</li>
  <li>Update images inside the <code>images/</code> folder.</li>
  <li>Modify colors, fonts, or layout in <code>style.css</code>.</li>
</ul>

<h2>📌 Notes</h2>
<ul>
  <li>Update <code>index.html</code> with your <strong>social media and contact details</strong>.</li>
  <li>Modify <code>scripts.js</code> if you want to add more interactivity.</li>
  <li>The contact form is connected to <strong>Google Sheets API</strong> for submissions. Update the script URL in the JavaScript file if needed.</li>
</ul>
